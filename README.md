# Mythix Vault

Захищена десктопна інформаційна система для управління персональними нотатками, тайм-менеджменту та безпечного зберігання даних. Розроблена в рамках дипломного проекту.

## 🛠 Технологічний стек
- **Мова програмування:** Python 3.10+
- **База даних:** SQLite
- **ORM:** SQLAlchemy
- **GUI Фреймворк:** CustomTkinter
- **Криптографія:** Cryptography (AES-256-CBC, PBKDF2)
- **Системні сповіщення:** Plyer

## 🔐 Ключові можливості
- **Абсолютна конфіденційність:** Увесь текстовий контент нотаток шифрується індустріальним стандартом AES. Паролі хешуються з використанням унікальної "солі".
- **Локальне зберігання:** Програма не потребує підключення до Інтернету, всі дані залишаються виключно на пристрої користувача.
- **Тайм-менеджмент:** Вбудована система дедлайнів з динамічним розрахунком часу.
- **Фонові сповіщення:** Асинхронний моніторинг часу з виведенням нативних Push-сповіщень Windows при наближенні дедлайну.
- **Організація даних:** Підтримка мультитегування, динамічної фільтрації та концепції "м'якого видалення" (Кошик).

## 🚀 Запуск з вихідного коду
1. Клонувати репозиторій:
   `git clone https://github.com/Igor7899/mythix-vault.git`
2. Встановити залежності:
   `pip install SQLAlchemy cryptography customtkinter plyer`
3. Запустити головний файл:
   `python main.py`

## 📦 Розгортання
Проект може бути скомпільований у самостійний `.exe` файл за допомогою утиліти PyInstaller (конфігурація вказана у файлі `MythixVault.spec`).
